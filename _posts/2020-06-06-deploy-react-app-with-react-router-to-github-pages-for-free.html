This article is to show you how to deploy React application to Github-pages quickly. In this tutorial, I will also show you how to configure React Router to your app.

I will show you how to create react app using create-react-appand deploy it to Github-pages.

For that, you need a Github account and Git installed on your computer. Also, make surenode.js and npm installed on your computer.

Step1:

Create a react application using create-react-app.

npx install --save create-react-app myapp
cd myapp
npm start

Your application is running on http://localhost:3000.

Step 2:

Install gh-pages pages as a dev-dependency

npm install --save-dev gh-pages

Step 3:

Open package.json file add "homepage”at the top level and add this value string http://{username}.github.io/{repo-name}

"homepage": "http://{your-username}.github.io/myapp",

add these lines to scripts

"scripts": {
    ...
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build"
}

Step 4:

create a public Github repository

run these commands (optional)

git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/{your-username}/myapp.git
git push -u origin master

Step 5:

Run npm run deploy. It will deploy your app to Github-pages

Your application is deployed to http://{username}.github.io/{repo-name}

Note:

If you are using React Router then BrowserRouter will not work for now. You can use HashRouter instead.

// import { BrowserRouter as Router } from 'react-router-dom';
import { HashRouter as Router } from 'react-router-dom'

If you want to user BrowserRouter then add this basename={process.env.PUBLIC_URL to Router
